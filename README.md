# About Ahmed Mohamed — Reveal.js CV

Responsive interactive CV built with React, Vite and Reveal.js.

## Navigation
- Left / right: move between CV sections
- Up / down: open details inside a section
- Esc: overview
- Swipe gestures work on touch devices
- The work-experience timeline stays visible and highlights the active role

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Notes
The employer marks are deliberately lightweight text-based brand markers. Replace each mark in `roles` with approved local logo assets if you have permission to use official brand artwork.
